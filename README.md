# Portfolio Website

A modern, responsive portfolio website built with Next.js 15, TypeScript, and Tailwind CSS. Deployed using Vercel, registered on Cloudflare.

---

## üöÄ Features

- **Responsive Design** - Fully optimized for mobile, tablet, and desktop devices
- **Dark Mode Support** - Automatic dark mode styling with Tailwind CSS
- **Dynamic Projects Page** - Fetches and displays public repositories from GitHub API
- **Interactive Contact Form** - Includes emoji-based captcha verification
- **Blog Section** - Category-based filtering (Tech, Economics, History)
- **Modern UI** - Clean, professional design with smooth transitions and hover effects
